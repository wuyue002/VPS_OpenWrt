# VPS_OpenWrt

This repo helps to deploy OpenWrt onto your VPS.<br>


**Steps**

1.  Upload OpenWrt firmware
2.  Run: `bash -c "$(wget -O- https://git.io/JGvno)"`

- Google Cloud
- Azure
- Vultr
- Virmach
